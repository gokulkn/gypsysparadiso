A premium, nature-focused website for **Gypsys Paradiso**, a camping and glamping retreat in Vattavada, Kerala.

## Features
- **Modern Design**: Glassmorphism, parallax effects, and smooth animations.
- **Interactive Map**: Custom "Journey" section with animated Jeep tracking.
- **Mobile Optimized**: Horizontal swipeable cards and touch-friendly navigation.
- **Performance**: Lazy loading, optimized assets, and clean code.

## Technologies
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript

## How to Host (GitHub Pages)
1.  Upload these files to a GitHub repository.
2.  Go to **Settings > Pages**.
3.  Select **Source: Deploy from a branch** and choose **main** (or master).
4.  Click **Save**. Your site will be live!
